# Лабораторная работа №1

## Цель работы:  
1. Разработать библиотеку для работы со структурой данных, указанной в индивидуальном задании, на языке C++.
2. Разработать тестовую программу, которая демонстрирует работоспособность реализованной библиотеки работы со структурой данных.
3. По результатам выполнения задания составить отчет.

## Задание:
AVL-дерево. Вставка. Удаление. Поиск. Поиск минимума, максимума, ближайшего большего и ближайшего меньшего.

## Ключевые понятия:
* **AVL-дерево** — это сбалансированное бинарное дерево поиска, в котором разница высот левого и правого поддеревьев любого узла не превышает 1.
* **Вставка** — операция добавления нового элемента в AVL-дерево с последующей балансировкой.
* **Удаление** — операция удаления элемента с сохранением баланса дерева.
* **Поиск** — нахождение узла с заданным ключом.
* **Поиск минимума и максимума** — определение наименьшего и наибольшего элементов в дереве.
* **Поиск ближайшего большего и меньшего** — нахождение элементов, ближайших к заданному по значению.

## Основные функции:
### Функция вставки узла
Добавляет новый элемент в дерево, сохраняя свойства бинарного дерева поиска. Проверяет балансировку и выполняет повороты при необходимости.

### Функция удаления узла
Удаляет узел из дерева, корректируя связи между элементами. После удаления проверяет балансировку и выполняет соответствующие повороты.

### Функция поиска узла
Находит элемент с заданным значением, проходя по дереву согласно правилам бинарного поиска.

### Функции поиска минимума и максимума
Определяют крайние элементы дерева (самый левый узел — минимум, самый правый — максимум).

### Функции поиска ближайшего большего и меньшего элемента
Ищут ближайший больший или меньший элемент относительно заданного ключа.
