# Лабораторная работа №2

## Цель работы:
1. Разработать программу на языке C++ для обработки множества элементов, представленного в строковом формате.
2. Реализовать функцию разбиения множества на элементы, учитывая вложенные множества.
3. Реализовать генерацию всех возможных перестановок элементов множества.
4. Записать результат в файл и вывести на экран.

## Задание:
Программа должна:
- Считать строку, содержащую множество в формате `{ a , b , c }`.
- Разобрать строку на отдельные элементы, сохраняя вложенные множества.
- Сгенерировать все возможные перестановки элементов.
- Вывести результат в консоль и записать в файл.

## Ключевые понятия:
* **Множество** — структура, содержащая уникальные элементы без определенного порядка.
* **Перестановка** — упорядоченный набор элементов, полученный перестановкой элементов множества.
* **Вложенные множества** — элементы множества, которые сами являются множествами, например `{d, e}`.

## Основные функции:
### Функция `stringToVectorOfSetElements`
Разбирает строковое представление множества и выделяет отдельные элементы, учитывая вложенные множества и пробелы.

### Функция `permute`
Генерирует все возможные перестановки элементов множества, используя рекурсию и метод обмена элементов.

### Функция `generatePermutations`
Вызывает `permute` и формирует вектор векторов строк, содержащий все возможные перестановки множества.

### Функция `findAllCombinations`
Считывает строку множества из файла, вызывает функции обработки и записи результатов.

## Вывод:
В результате работы была разработана программа, способная разбирать строковое представление множества, генерировать его перестановки и сохранять их в файл. Работа позволила изучить разбор строк, обработку вложенных структур и рекурсивные алгоритмы генерации перестановок.
