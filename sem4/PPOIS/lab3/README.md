# Minesweeper (Lab 3)

Графическое приложение «Сапёр» на `pygame`.

## Запуск
1. Установить зависимости:
   ```bash
   pip install -e .
   ```
2. Запуск:
   ```bash
   python -m minesweeper
   ```
   или
   ```bash
   minesweeper
   ```

## Меню
- **начать игру** → выбор режима и сложности.
- **таблица рекордов** → рекорды по режимам и сложностям.
- **справка** → правила и управление.
- **выход** → закрыть приложение.

## Режимы
- **Классика**: без лимита времени.
- **На время**: у каждого уровня есть лимит, по истечении — поражение.

## Рекорды
- Рекорд определяется минимальным временем прохождения.
- Ведётся отдельно для каждого режима и сложности.
- Файл: `config/records.json`.

## Конфигурации
Все основные настройки вынесены во внешние файлы:
- `config/game.json` — уровни, лимиты времени, размеры, цвета, пути к звукам.
  - Текст подсказок можно менять в `ui.hints`.
  - Лимиты времени для режима «На время» — в `modes.timed.time_limits`.
- `config/help.md` — текст справки.
- `config/records.json` — таблицы рекордов.

## Управление
- ЛКМ: открыть клетку.
- ПКМ: поставить/убрать флаг.
- ЛКМ по открытому числу (при достаточном числе флагов) — открыть соседние клетки.
- Enter: подтверждение.
- Esc: назад / выход в меню.

## Структура проекта
- `src/minesweeper/app.py` — основной цикл, экраны, отрисовка.
- `src/minesweeper/board.py` — логика поля.
- `src/minesweeper/game_state.py` — состояние игры.
- `src/minesweeper/records.py` — таблицы рекордов.
- `src/minesweeper/audio.py` — генерация/проигрывание звуков.
- `src/minesweeper/utils.py` — утилиты (JSON, формат времени, переносы текста).
- `config/` — конфиги и справка.
- `assets/sounds/` — звуки (создаются при первом запуске, если отсутствуют).
